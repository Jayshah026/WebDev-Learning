# Basics of HTML 

## Tags and their uses 

1. <title> </title> : For naming the website so user can identifie it.
2. <p> </p> : By this tag we can make multiple paragraphs. 
3. <h1> </h1> : This is the main heading and beside this there are 6 other headings form h1 to h6.
4. <storng> </strong> : This makes text bold.
5. <em> </em> : This makes text italic.

## List Types

1. Unorderd List :

- To use this we have to first use <ul> </ul> and in between them use <li> </li> if the order is not matter.

2. Orderd List :

- To use this we have to first use <ol> </ol> and in between them use <li> </li> if order matters.

=> The main difference is that if the order not matter then use <ul> and if matters then use <ol> and between them use <li> to make the list. 


## Important

1. In the HTML we need to add <!DOCTYPE html> at the starting of the code.
2. <html> </html> : This contain all the data inside it.
3. <body> </body> : We can add data which can be shown into the website. 
4. <head> </head> : We can add data which works in the background.
